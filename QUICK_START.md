# 🚀 Quick Start Guide - ResumeIQ

## 5-Minute Setup

### Step 1: Get Groq API Key (2 minutes)
1. Visit [console.groq.com](https://console.groq.com)
2. Click "Sign In" or "Get API Keys"
3. Create free account (email verification required)
4. Copy your API key from the dashboard
5. Keep it safe - you'll need it in the app

### Step 2: Open the App (1 minute)
**Option A: Direct in Browser**
- Download/Clone this repository
- Open `index.html` in your browser

**Option B: Local Server (Recommended)**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server

# Then visit: http://localhost:8000
```

### Step 3: Configure API Key (1 minute)
1. Click ⚙️ **Settings** button (top right)
2. Paste your Groq API key
3. Click "Save API Key"
4. Wait for verification (should show ✓ success)

### Step 4: Analyze Resume (1 minute)
1. Paste your resume in the text area
2. (Optional) Paste job description for job matching
3. Click "Analyze Resume"
4. Wait for AI analysis (~5-10 seconds)
5. Review results and recommendations

## 🔑 Getting Your Groq API Key

### Free Tier Includes:
- ✅ 30 requests per minute
- ✅ 2,000 tokens per request

- ✅ No credit card required (for first tier)

### Steps:
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up with email or OAuth
3. Verify email
4. Navigate to "API Keys" section
5. Click "Create API Key"
6. Copy the key
7. Paste in ResumeIQ Settings

## 🆘 Quick Troubleshooting

| Problem | Solution |

## 💡 Tips for Best Results

### Resume Tips
- ✅ Include measurable achievements (numbers, percentages)
- ✅ Use industry keywords for your role
- ✅ Highlight technical skills prominently
- ✅ Include projects with brief descriptions
- ✅ Add relevant certifications

### Job Matching Tips
- ✅ Copy entire job description from LinkedIn/Indeed
- ✅ Include requirements, responsibilities, and benefits
- ✅ Tool will highlight exactly what you're missing

## 🎓 What is ATS?

**ATS = Applicant Tracking System**

It's software that companies use to:
- Parse and scan resumes
- Extract keywords and qualifications
- Filter candidates automatically
- Rank candidates by match score

ResumeIQ helps you optimize your resume to pass ATS screening!
