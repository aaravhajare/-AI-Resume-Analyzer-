## 🚀 Quick Start (5 Minutes)

### Step 1: Get Groq API Key (2 min)
```
1. Visit: https://console.groq.com
2. Sign up (free)
3. Go to API Keys section
4. Create and copy your API key
```

### Step 2: Run Application (1 min)
```bash
# Option A: Python
python -m http.server 8000

# Option B: Node.js
npx http-server -p 8000

# Then open: http://localhost:8000
```

### Step 3: Configure API (1 min)
```
1. Click ⚙️ Settings
2. Paste your Groq API key
3. Click "Save API Key"
4. See ✓ Confirmation
```

### Step 4: Analyze Resume (1 min)
```
1. Paste your resume
2. (Optional) Paste job description
3. Click "Analyze Resume"
4. Wait 5-10 seconds for analysis
5. Review comprehensive feedback
```
